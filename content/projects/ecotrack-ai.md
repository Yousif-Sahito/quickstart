## Overview

EcoTrack AI is an **AI-powered sustainability application** that helps users understand and reduce their environmental impact. It analyzes lifestyle inputs and generates personalized recommendations for lowering carbon emissions.

The system combines a mobile-friendly frontend with a backend API and ML-driven insights to deliver actionable sustainability guidance.

---

## ⬇️ Download / Access

EcoTrack AI is designed as a cross-platform application with a dedicated API backend.

**Platform Support**

- Mobile app via React Native
- REST API backend for data processing
- MySQL database for user and footprint records

> EcoTrack AI focuses on **readability and actionable insights**. Users receive clear breakdowns of their footprint categories and practical steps to improve.

---

## 🏗️ System Architecture

The project follows a **client-server architecture** separating user interaction from data processing and AI analysis.

### Architectural Layers

1. **Frontend Layer (React Native)**
   - User onboarding and input forms
   - Dashboard with footprint visualizations
   - Recommendation display

2. **Backend Layer (Node.js + Python)**
   - REST API for user data and calculations
   - Python services for AI/ML analysis
   - Authentication and session management

3. **Data Layer (MySQL)**
   - User profiles and history
   - Footprint records and category breakdowns

---

## 🔎 Core Features

- **Footprint Analysis:** Calculates emissions based on transport, energy, and lifestyle inputs.
- **AI Recommendations:** Suggests personalized actions to reduce environmental impact.
- **Progress Tracking:** Lets users monitor improvements over time.
- **Category Breakdown:** Visualizes which areas contribute most to total footprint.
