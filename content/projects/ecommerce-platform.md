## Overview

A full-featured **e-commerce platform** for a modern clothing store. The application includes user authentication, product catalog, shopping cart, Stripe payment integration, and an admin dashboard for inventory management.

Built with scalability and clean separation of concerns in mind.

---

## ⬇️ Features

- User registration and login
- Product browsing with categories and filters
- Shopping cart and checkout flow
- Stripe payment processing
- Admin dashboard for product and order management

---

## 🏗️ System Architecture

1. **Frontend (React)** — Product pages, cart, checkout UI
2. **Backend (Node.js + Express)** — REST API, auth, payment webhooks
3. **Database (MySQL)** — Users, products, orders, inventory
