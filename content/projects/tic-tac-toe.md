## Overview

A **web-based Tic Tac Toe game** where two players take turns. The game detects winners or draws and includes a reset button to play again.

A clean front-end project demonstrating DOM manipulation, game state management, and responsive UI design.

---

## 🔎 Core Features

- Two-player turn-based gameplay
- Win and draw detection
- Reset button for new games
- Responsive layout

---

## 🧠 Tech Stack

- HTML5 for structure
- CSS for styling and layout
- Vanilla JavaScript for game logic
